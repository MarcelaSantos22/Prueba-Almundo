Prueba Técnica
=====================

## Getting Started

Primero: Servidor

Entrar a la carpeta

```bash
$ cd almundo-back
```

Instalar npm devDependencies

```bash
$ npm install
```

Correr el servidor en el back
```bash
$ npm run start
```

Segundo: Aplicacion web

Entrar a la carpeta

```bash
$ cd almundo-web
```

Instalar npm devDependencies

```bash
$ npm install
```

Correr aplicación
```bash
$ npm start
```
